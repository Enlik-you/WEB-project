# HiTrade

HiTrade is a full-stack wholesale marketplace for Korean products in the CIS region.

## Description
This project is being developed as a web platform for buying and selling Korean goods in bulk.  
The front-end is built with Angular, and the back-end is built with Django + DRF.

## Features
- Product catalog
- Categories
- Login and registration
- JWT authentication
- Orders
- API integration
- Routing
- Forms with ngModel
- Error handling

## Front-End
- Angular
- TypeScript
- HTML
- CSS

## Back-End
- Django
